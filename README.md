# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 임의의 1-9까지 서로다른 수 생성
* 생성된 수 배열에 넣기.
* 플레이어에게 1-9 서로다른 수 입력 받기.
* 입력 값 검증
* 검증 후 결과 값 리턴
* 정답 시 게임 종료
* 에러 시 앱 종료
* 게임 종료시 게임 다시하기 및 앱 종료.
